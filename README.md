### 组织结构
```lua
mall
├── document -- 项目相关文件
├── mall-api -- 接口模块，为网页、APP等前端提供接口
├── mall-backend-api 后台接口模块
├── mall-common -- 通用模块，提供通用代码
├── mall-pojo -- 实体模块，提供数据库实体对象
├── mall-service -- 业务模块，提供商城业务代码
├── mall-util -- 工具模块，提供相关工具类
```
